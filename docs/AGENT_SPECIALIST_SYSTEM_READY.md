## 🚀 How to Use Them

### Create Sessions
1. Create 4 new OpenClaw sessions (one per agent)
2. Load each SOUL.md file as context
3. Use Claude Opus as the model
4. Test with simple queries

### Talk Naturally
```
You: "Scout, what psychology topics should we focus on this month?"
Scout: [Deep research response]

You: "Finch, what's our budget situation?"
Finch: [Financial analysis]

You: "Pulse, which of our videos are working best?"
Pulse: [Performance analysis]

Psyche: [Coordinated strategy]

```

## 🔄 Comparison: Specialist Agents vs. Scheduled Agents

| Feature | Specialist (Opus) | Scheduled (Haiku) |
|---------|-------------------|-------------------|
| **Model** | Opus (sophisticated) | Haiku (lightweight) |
| **Interface** | Natural conversation | Telegram commands |
| **Expertise** | Deep & professional | Task-focused |
| **Use Case** | Strategy, analysis, decisions | Routine automation |
| **Cost** | Higher | Lower |
| **Interaction** | Chat naturally | Request & execute |
| **Best For** | Thinking things through | Getting things done |

**TL;DR:** Use specialists for strategy/analysis, scheduled agents for routine tasks.
